## Tailwind CSS 4.2 Setup

- **Version**: `tailwindcss@4.2.0` + `@tailwindcss/postcss@4.2.0` (CSS-first config, no `tailwind.config.js`)
- **PostCSS**: `postcss.config.mjs` uses `@tailwindcss/postcss` plugin
- **Global CSS**: `src/app/globals.css` — all theme config lives here

### Theme System
Tokens are defined as CSS custom properties in `:root` (light) and `.dark` (dark), then mapped to Tailwind utilities via `@theme {}`.

| Token | Utility | Use for |
|---|---|---|
| `--background` | `bg-background` | Page background |
| `--foreground` | `text-foreground` | Body text |
| `--card` / `--card-foreground` | `bg-card`, `text-card-foreground` | Card surfaces |
| `--primary` / `--primary-foreground` | `bg-primary`, `text-primary-foreground` | Primary actions |
| `--secondary` / `--secondary-foreground` | `bg-secondary`, `text-secondary-foreground` | Secondary elements |
| `--muted` / `--muted-foreground` | `bg-muted`, `text-muted-foreground` | Subtle backgrounds/text |
| `--accent` / `--accent-foreground` | `bg-accent`, `text-accent-foreground` | Hover states |
| `--destructive` | `text-destructive`, `bg-destructive` | Errors / danger |
| `--border` | `border-border` | Borders |
| `--input` | `border-input` | Input fields |
| `--ring` | `ring-ring` | Focus rings |

### Dark Mode
- **Class-based**: add `.dark` to `<html>` for manual toggle (`@custom-variant dark (&:where(.dark, .dark *))`)
- **OS fallback**: `@media (prefers-color-scheme: dark)` applies dark tokens when no `.dark`/`.light` class is set
- **Prevent hydration mismatch**: `<html suppressHydrationWarning>` is set in `src/app/layout.tsx`
- **Light override**: add `.light` class to `<html>` to force light mode even when OS is dark

### Font tokens
- `--font-sans` → Geist Sans (from `--font-geist-sans` CSS var set by Next.js font)
- `--font-mono` → Geist Mono (from `--font-geist-mono` CSS var set by Next.js font)
- Use as: `font-sans`, `font-mono` Tailwind utilities

### Border radius scale
All radii are relative to `--radius` (default `0.5rem`): `rounded-xs`, `rounded-sm`, `rounded-md`, `rounded-lg`, `rounded-xl`, `rounded-2xl`, `rounded-3xl`
